Conheça meu trabalho:

Futebol Placar
Aplicativo Android que fornece ao usuário informações dos principais campeonatos de futebol, desenvolvi e publiquei na Play Store.
APIs de informações relacionadas a esportes tem um alto custo financeiro, para contornar esse custo criei um script utilizando Python que realiza Web Scraping em diversos sites do nicho e alimenta o banco de dados Firestore.
Implementei a sincronização dos dados com firestore visando evitar sobrecarregar o consumo de dados do usuário e evitar um alto custo nos servidores Firebase.
Criei a interface utilizando Compose, utilizei Navigation do kit Jetpack Compose para controlar a navegação e entregar uma boa experiência para o usuário.
Apliquei os fundamentos MVVM utilizando ViewModel, facilitando a manutenção do código.
Introduzi banners e anúncios intersticiais usando AdMob, focando sempre em usar boas práticas.
Solo Boss - NW
Aplicativo Android de informações referente ao jogo New World, desenvolvi e publiquei na Play Store.
Criei a interface utilizando Compose, para consistência e consulta do banco de dados foi utilizado SQLite, Room.
Para manter a lógica de negócios separada da interface do usuário utilizei ViewModel, aplicando o padrão MVVM.
Implantei banners e anúncios Intersticiais via AdMob, mantendo boas práticas.
Localiza CEP
Aplicativo Android que possibilita a consulta de informações a partir do CEP informado, desenvolvi e publiquei na Play Store.
As informações que são passadas para o usuário são obtidas a partir de uma API pública, utilizei Retrofit para lidar com o consumo da API.
Utilizei Compose para criação de interface e Navigation do kit Jetpack Compose para controlar a navegação e entregar uma boa experiência para o usuário.
Visando manter o padrão MVVM, utilizei o ViewModel para separar a lógica de negócios da interface do usuário.
Optei por usar banner fixo utilizando AdMob

Saiba mais sobre mim +

  Sou um filho do coração pulsante do Mato Grosso do Sul. Aos 21 anos, meu destino me levou para as terras acolhedoras do interior de Pernambuco, mais precisamente Limoeiro. Aqui, nesses últimos 11 anos, encontrei mais que um lar; encontrei minha alma gêmea, Katarine, com quem compartilho uma jornada de amor e cumplicidade que já se estende por uma década.

  Minha paixão por animais transcende barreiras financeiras. Mesmo em tempos de recursos escassos, ergui um refúgio para gatos abandonados, um oásis onde doze vidas peludas encontraram abrigo e cuidado. Afinal, o amor por esses seres de patas e bigodes é uma força inesgotável que ilumina meus dias.

  Ao longo da minha jornada, trilhei caminhos profissionais diversos, sempre perseguindo uma remuneração que sustentasse meus sonhos. No auge dos meus 31 anos, tomei uma decisão que reverberou desde a criança sonhadora que eu era: decidi dar vida aos códigos. Para mim, a programação não é apenas um ofício, mas uma forma de expressar arte. É uma dança intricada entre lógica e criatividade, uma sinfonia de zeros e uns que ecoa a realização de um sonho outrora tido como impossível.

  Dentro desse universo de linhas de código, encontrei meu refúgio, meu santuário criativo. Programar é mais que uma profissão; é uma paixão, uma jornada de descoberta constante e um veículo para transformar sonhos em realidade. Quero não apenas sobreviver, mas viver plenamente dessa arte que aquece meu coração e dá sentido às linhas que tecem minha trajetória. Com o firme propósito de transformar minha paixão por programação em uma carreira promissora, busco agora uma oportunidade como desenvolvedor júnior.
